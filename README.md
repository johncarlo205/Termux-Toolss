# Termux-Toolss
Automatic program installer for Android [ONLY FOR TERMUX]

# Only for Termux Users


# Installation
```
git clone https://github.com/tejhacker/Termux-Toolss
cd $HOME/Termux-Toolss
chmod +x All-Toolss.py
sh $HOME/Termux-Toolss/setup
```
